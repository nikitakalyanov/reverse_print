# reverse_print_linked_list

A very simple recursive function to print (1-way) linked list of the following structure:
```python
some_list = {
  'value': 1,
  'next': {
    'value': 2,
    'next': {
      'value': 3,
      'next': {
        'value': 4,
        'next': None,
      },
    },
  },
}
```
## Usage

`reverse_print(some_list)`
